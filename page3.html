Curriculum: page 3, more pictures, please!

We've seen how we can use JavaScript to move content around on the page by changing the CSS. We can even react to the user by listening to events.

But what if you want to add more photos to you slide show? You could do it by adding more <img> tags to the HTML, which is not a bad idea.

We could also be very clever, and use JavaScript to add images. We can add and delete and edit the content of the document, just like we can change the CSS.

Using JavaScript just to be clever is not a good idea for "real" web pages. But being clever is a fun way to learn. Plus, there are situations where editing the content using JavaScript is a good idea.

Add content to the document using JavaScript

Goal

See how to edit the content of the document with JavaScript DOM properties and methods.

Instructions

Visit your slide show in Chrome, and open the console.

Remember, since we have our slideshow.js running, we have a reference to the "file role" in the variable called filmroll. In the console, type:

filmroll.innerHTML

Is the innerHTML property of filmroll a read-only value? Find out by trying to set to to some other value:

filmroll.innerHTML = "Remember, strings need quotes."

You can also use numbers, and do math, it you wanted:

filmroll.innerHTML = 4 + 3 * 111 + 1000

Try it with a string of HTML:

filmroll.innerHTML = "<h1>Whoa!</h1> <p>Check it out.</p>"

We call add HTML to the HTML that is already there using the += operation:

filmroll.innerHTML += "<p>Yeah! Lots of potential.</p>"

Try this:

filmroll.innerHTML = "<img src='http://tif.ca/hello.jpg'>"

Explanation

One of the easiest way to create content in a web page using JavaScript is to set the innerHTML of an element to something new. Since HTML is just text, you can set the innerHTML to any text and it is rendered by the browser as HTML.

We can read the innerHTML of an element as well we write to it, which means we can add the new HTML to the existing innerHTML to add, rather than replace, content.

This technique is not the only way to add content to the document, but for today, we will only use this one.
